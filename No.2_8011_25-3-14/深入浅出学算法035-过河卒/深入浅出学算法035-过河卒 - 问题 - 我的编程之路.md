问题 F: 深入浅出学算法035-过河卒

### 问题 F: 深入浅出学算法035-过河卒

<sub>[命题人 : <span id="creator"><a href="http://47.96.116.66/userinfo.php?user=admin">admin</a></span>]</sub>

时间限制 : 1.000 sec  内存限制 : 128 MB  
  

#### 题目描述

棋盘上A点有一个过河卒，需要走到目标B点。卒行走的规则：可以向下、或者向右。同时在棋盘上的任一点有一个对方的马（如C点），该马所在的点和所有跳跃一步可达的点称为对方马的控制点（如图中的C点和P1，P2，……，P8）。卒不能通过对方马的控制点。棋盘用坐标表示，A点(0,0)、B点(n, m) (n,m为不超过20的整数),同样马的位置坐标是需要给出的，C≠A且C≠B。现在输入B点坐标和C点的坐标，要你计算出卒从A点能够到达B点的路径的条数。  
![](http://47.96.116.66/upload/image/20181109/20181109114442_24430.png)  

#### 输入

输入B点坐标和C点的坐标

#### 输出

卒从A点能够到达B点的路径的条数。

#### 样例输入 [Copy](http://47.96.116.66/)

```
4  8  2  4
```

#### 样例输出 [Copy](http://47.96.116.66/)

```
0
```
